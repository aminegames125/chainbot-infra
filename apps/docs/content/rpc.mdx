# RPC & network

## Network details

| Parameter | Value |
|---|---|
| Chain ID | `13371` |
| RPC HTTP | `https://chainbot.animeos.dev/rpc` |
| RPC WS | `wss://chainbot.animeos.dev/ws` |
| Explorer | `https://chainbot.animeos.dev/explorer` |
| Currency | `COIN` |
| Block time | `~12 seconds` |

## Add to MetaMask

Click **Add Network** on the landing page, or add manually:

```json
{
  "chainId": "0x3439",
  "chainName": "ChainBot",
  "rpcUrls": ["https://chainbot.animeos.dev/rpc"],
  "nativeCurrency": { "name": "COIN", "symbol": "COIN", "decimals": 18 },
  "blockExplorerUrls": ["https://chainbot.animeos.dev/explorer"]
}
```

## Rate limits

The public RPC is rate-limited to **100 requests per minute per IP** via Nginx.
For high-frequency access (bots, scripts), self-host a second node.
