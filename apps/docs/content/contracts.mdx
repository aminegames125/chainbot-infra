# Smart contracts

All contracts are deployed on Chain ID **13371**.

## Contract addresses

After deploying with `pnpm contracts:deploy`, all addresses are saved to
`packages/contracts/deployments.json` and auto-imported by the bot and web app.

## Contract list

| Contract | Purpose |
|---|---|
| `PriceOracle` | On-chain USD prices for all assets |
| `MintController` | Daily claims + work rewards (all mints go through here) |
| `VaultManager` | Stablecoin collateral vaults |
| `MemeCoinFactory` | Deploys community meme tokens |
| `NFTCollection` | ERC-721 badges and shop items |
| `MinerRegistry` | Tracks real miner hashrates |
| `UniswapV2Factory` | Creates trading pairs |
| `UniswapV2Router02` | Executes swaps |
| `WETH9` | Wrapped COIN for DEX compatibility |

## Transparency

Every state-changing function emits an on-chain event.
The dashboard's Transparency tab reads these events live and flags any mint
that does not originate from `MintController` in **red**.
