# Getting started

## Requirements

- VPS with 6 GB RAM (Ubuntu 22.04 recommended)
- Docker + Docker Compose v2
- pnpm 9+, Node.js 20+
- A domain pointed at your server

## One-server deploy

### 1. Clone and install

```bash
git clone https://github.com/yourorg/chainbot
cd chainbot
cp .env.example .env   # fill in all values
pnpm install
```

### 2. Init the Geth chain (first time only)

```bash
docker run --rm -v ./packages/geth:/geth ethereum/client-go:v1.13.14 init /geth/genesis.json
```

### 3. Create treasury wallet (first time only)

```bash
docker run --rm -v ./packages/geth/keystore:/keystore ethereum/client-go:v1.13.14 \
  account new --keystore /keystore
```

Copy the printed address into `.env` as `TREASURY_ADDRESS`.

### 4. Start the chain and deploy contracts

```bash
docker compose up -d geth postgres
sleep 15
pnpm contracts:deploy
```

### 5. Start everything

```bash
docker compose up -d
```

The bot is live, the website is at port 3000, and docs are at port 3001.
