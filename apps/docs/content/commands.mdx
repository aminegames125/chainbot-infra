# Discord commands

## Wallet
| Command | Description |
|---|---|
| `/wallet create` | Generate a new wallet. Private key sent via DM. |
| `/wallet import 0x...` | Link an external address (view-only). |
| `/balance` | Show all token balances + USD values. |
| `/send @user 100` | Transfer COIN to another user. |

## Mining & daily
| Command | Description |
|---|---|
| `/daily` | Claim your daily COIN reward (24h cooldown). |
| `/miner download` | Get the MinerCLI for Windows, Mac, or Linux. |
| `/hashrate` | View your mining contribution. |
| `/leaderboard miners` | Top 10 miners by hashrate. |

## Tokens & DEX
| Command | Description |
|---|---|
| `/token create NAME TICKER` | Deploy a meme coin (costs COIN). |
| `/token info $TICKER` | Price, holders, market cap. |
| `/swap $A $B 100` | Swap tokens via Uniswap V2. |
| `/price $TICKER` | Live USD price. |
| `/liquidity add $TOKEN` | Add liquidity, earn fees. |

## Stablecoins
| Command | Description |
|---|---|
| `/vault open 100` | Lock COIN as collateral. |
| `/vault mint sUSD 50` | Borrow a stablecoin. |
| `/vault repay 50` | Repay and unlock collateral. |
| `/vault status` | Collateral ratio + liquidation price. |

## Economy
| Command | Description |
|---|---|
| `/work` | Random job reward (1h cooldown). |
| `/gamble 100` | 50/50 win or burn. |
| `/shop` | Buy NFTs and badges. |
| `/inventory` | Your owned assets. |
| `/leaderboard rich` | Top 10 COIN holders. |
